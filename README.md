# Disease Prediction from Medical Data

## Objective
Predict the possibility of diseases based on structured medical data.

## Dataset
Breast Cancer Wisconsin Diagnostic Dataset (UCI Machine Learning Repository)

## Approach
- Logistic Regression
- Random Forest

## Key Steps
- Data preprocessing and cleaning
- Feature scaling
- Model training and evaluation

## Evaluation Metrics
- Precision
- Recall
- F1-Score
- ROC-AUC

## Result
Random Forest achieved the best overall performance for disease prediction.
